# 🚀 ResumeAI – ATS Resume Analyzer

## 🔥 Features
- Resume upload (PDF/DOCX)
- Skill extraction
- Match score calculation
- Domain detection
- Smart suggestions
- Resume bullet generation
- Download improved resume

## 🌐 Live Demo
https://resume-ai-fr3jhjb3dnrup9vqkjfbpt.streamlit.app/

## 🛠️ Tech Stack
- Python
- Streamlit
- NLP (basic)
- PyPDF2, python-docx

## 💡 Description
This project simulates an ATS system used in recruitment. It helps users evaluate and improve their resumes based on job descriptions.
